# Data Engineering Copilot: Agent Guide

This guide describes how to interact with the 20+ specialized AI agents inside the Data Engineering Copilot.

## Core Data Agents
1. **SQL Agent**: Generates CDC queries, UPSERT/MERGE scripts, and reconciliation reports.
2. **Databricks Agent**: Implements slowly changing dimensions (SCD Type 1/SCD Type 2) and optimizations using PySpark.
3. **ADF Agent**: Generates watermark copy pipeline definitions and ADF dynamic expressions.
4. **Dataverse Agent**: Maps relational schemas to Dataverse entities and Power Query M ingestion flows.

## Advanced Data Agents (Batch 3)
5. **Data Quality Agent**: Builds Great Expectations suites and Soda Core YAML checks.
6. **dbt Agent**: Generates dbt staging models, snapshots, tests, and custom Jinja macros.
7. **Airflow Agent**: Designs Apache Airflow DAGs with task groups, file sensors, and dynamic flow execution.
8. **Terraform IaC Agent**: Builds modular Terraform files for multi-cloud data infrastructure (AWS, Azure, GCP).
9. **Data Governance Agent**: Generates secure RBAC policies and automated column masking/PII classification rules.
10. **Cost Optimization Agent**: Computes optimal cluster sizing parameters and storage lifecycle rules.
11. **Data Migration Agent**: Builds platform migration assessments, type conversion mappings, and rollback plan runbooks.
12. **Observability Agent**: Defines metrics SLIs/SLOs frameworks and Prometheus pipeline failure alert thresholds.

## Specialized Agents (Batch 4)
13. **Data Catalog Agent**: Document dictionaries compiler and Mermaid lineage graphs renderer.
14. **Testing Agent**: Mock insert queries generator and query validation comparison test templates.
15. **Code Review Agent**: Review code segments for anti-patterns and performance bottlenecks.
16. **API Integration Agent**: Connect APIs safely using rate-limiting, pagination, and Fast API webhooks handlers.
17. **Streaming Agent**: Designs stream-stream / stream-static windows in Spark and Apache Flink.
18. **MLOps Data Agent**: Ingestion setup for Feast Feature Store and data drift KS-tests monitors.
